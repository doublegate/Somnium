Play together with friends in real-time! Our new multiplayer system features:

### Run Your Own Server

For AI-generated worlds and full multiplayer:

```bash
# Clone the repository
git clone https://github.com/doublegate/Somnium.git
cd Somnium

# Frontend setup
npm install
npm start

# Backend setup (optional, for multiplayer/cloud)
cd server
npm install
cp .env.example .env
# Edit .env and add your configuration
npm run start:all
```

Full setup guide: [Server Deployment Documentation](https://github.com/doublegate/Somnium/blob/main/docs/server-deployment.md)
